# 🛡️ 守护进程速查

## Mac mini (launchd)

| Label | 跑什么 | 启动时机 | 文件 |
|---|---|---|---|
| `local.sunny.hermes-gateway` | Hermes gateway + 微信 iLink | RunAtLoad | `~/Library/LaunchAgents/local.sunny.hermes-gateway.plist` |
| `local.sunny.daily-cleanup` | 每日 3:30 清理系统垃圾 | StartCalendarInterval 03:30 | `~/Library/LaunchAgents/local.sunny.daily-cleanup.plist` |

### 常用命令

```bash
# 列出所有
launchctl list | grep local.sunny

# 手动跑一次
launchctl start local.sunny.daily-cleanup

# 重启 Hermes gateway
launchctl kickstart -k local.sunny.hermes-gateway

# 查看日志
tail -f ~/.hermes/logs/launchd_hermes_gateway.out.log
tail -f ~/.hermes/logs/launchd_hermes_gateway.err.log
```

## dgx spark (systemd user)

| Unit | 跑什么 | 启动时机 |
|---|---|---|
| `hermes-gateway.service` | Hermes gateway | 启动时 + 失败重启 |
| `comfyui.service` | ComfyUI 0.25.1 | 启动时 + 失败重启 |

### 常用命令

```bash
ssh spark

# 状态
systemctl --user status hermes-gateway
systemctl --user status comfyui

# 启停
systemctl --user restart hermes-gateway
systemctl --user restart comfyui

# 看日志
journalctl --user -u comfyui -f
tail -f /home/new/git/ComfyUI-0.25.1/comfyui.log
```

### Lingering

```bash
# 检查 Linger
loginctl show-user new | grep Linger  # Linger=yes

# 启用（无需 sudo，新版 systemd 行为）
loginctl enable-linger new
```
